Turbo Examples

1) Replace content in a turbo frame

By default, any link from within a Turbo Frame will look for a matching frame in the response and load it in the frame. For reference, see what a normal request to the same page looks like.

Click

2) Load content in a turbo frame from a link outside the frame

When the link to trigger a response is outside the turbo frame, use data-turbo-frame="name-of-frame" to have the response load in a frame

Click Content will be replaced

3) Break out of a turbo frame

Use data-turbo-frame="_top" on a link to do a normal page visit instead of loading content in the frame

Break out of frame

4) Update the page url

Use data-turbo-action="advance" to update the page url with of a link followed in the frame

Click

5) Eager load a frame from a src url

Use src="/five.html" to fetch content for a frame on page load. Notice the frame blinks to show the "Loading..." first. Check the network tab to see the request for five.html. Change the src to a page that doesn't exist to better see the loading state.

6) Lazy load a frame from a src url

Use src="/six.html" and loading="lazy"to fetch content for a frame only when the frame comes into view. To test: minimize the browser size so this example is off screen. With the network tab open, scroll down and notice that six.html is only loaded when this frame comes into view.
